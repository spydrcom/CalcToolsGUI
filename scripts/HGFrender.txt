

//* CalcLib version of HGF render


A = (1,2,3);		// A will be Array of coefficients

// rho function will be Pochammer product of coefficient array
!! rho(A,nA,n) = PI [ 1 <= i <= nA ] ( A # ( i - 1 ) /#+ n ) 

// lambda will pass array with length to rho
!! lambda(A,n) = rho ( A , LENGTH A , n ) 

// mu is quotient of numerator and denominator array expansion
!! mu (An,Ad,n) = lambda ( An , n ) / lambda ( Ad , n ) 

// hyper-gometric function expanded as infinite array
!! F ( An, Ad, z ) = SUMMATION [ 0 <= n <= INFINITY ] ( mu (An,Ad,n) * (z^n / n!)  ) 

// render the chain of functions

RENDERF F
RENDER "The Hyper-Geometric Function as an infinite series" TOP

RENDERF mu
RENDER "For each term of the series the ratio of Pochammer raising products of the coefficients is computed" TOP

RENDERF lambda
RENDER "Pass the coefficient array and the length to the Pochammer product" TOP

RENDERF rho
RENDER "The product of the Pochammer of each of the coefficient array components" TOP
