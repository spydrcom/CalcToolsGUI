n = 2

configure Bessel form "integral" kind "K" method "CTA" alpha "n" delta "0.001" terms "20" precision "1E-4" description "Bessel function K (n)" ; instance Kn Bessel

!! Y(x) = Kn (x)

configure dydx symbol "d" order "1" function "Y" variable "x" run "dx" description "dY/dx" ; instance dYdx1 dydx
!! KnDE(x) = x^2 * dYdx2() + x * dYdx1() - (x^2 + n^2) * Y(x)
GRAPH LIM 0.2 [0.2 < x < 5 <> 0.1] ( KnDE x )
ENTITLED Differential Equation error plot for Kn computed based on inline configuration
